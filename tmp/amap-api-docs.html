API

为开发者提供HTTP接口，可通过这些接口使用各类型的地理数据服务

地理/逆地理编码 API HOT 搜索 通过 HTTP/HTTPS 协议访问远程服务的接口，提供结构化地址与经纬度之间的相互转化的能力 路径规划 API HOT 导航 以HTTP形式提供的步行、公交、驾车查询及行驶距离计算接口，返回JSON 或 XML格式的查询数据，用于实现路径规划功能的开发 搜索 API HOT 搜索 提供多种查询POI信息的能力，其中包括关键字搜索、周边搜索、多边形搜索、ID查询四种筛选机制 输入提示 API HOT 搜索 根据用户输入的关键词查询返回建议列表 行政区域查询 API 搜索 根据用户输入的搜索条件可以帮助用户快速的查找特定的行政区域信息 静态地图 API 地图 通过返回一张地图图片响应HTTP请求，使用户能够将高德地图以图片形式嵌入自己的网页中 IP 定位 HOT 搜索 根据用户输入的 IP 地址，能够快速的帮用户定位 IP 的所在位置 坐标转换 API 地图 将用户输入的非高德坐标（GPS坐标、mapbar坐标、baidu坐标）转换成高德坐标 天气查询 API 搜索 根据用户输入的adcode，查询目标区域当前/未来的天气情况 轨迹纠偏 API 导航 根据给定的坐标点、车辆的方位角以及行驶速度，将用户的轨迹纠偏到路上，从而返回用户实际驾车经过的道路坐标 云图服务 API 地图 为您提供位置数据存储、检索、展现服务，助您快速搭建基于业务数据的网站或APP

JS API

基于第四代 Web 地图渲染引擎， 以 WebGL 为主要绘图手段提供Web地图服务

JavaScript API 2.0 HOT 地图 完美适配各种浏览器,包含地图显示/搜索/路线规划/地址解析/一键调起高德地图 APP 等功能 地图组件 地图 专为手机浏览器定制,通过URL形式调用,简单快速地实现位置标注/路线规划/周边搜索/地图选址的功能 URI API 地图 适用于原生 APP 以及手机浏览器,通过URL形式调用.包含标注/路径规划/ POI 检索/路线规划等功能 地铁图 JS API 地图 适用于开发地铁图 H5 页面,包含地铁线路展示/路线查询/站点标注以及路线规划功能

Android 平台

开发者可以轻松地在自己的 Android 应用中加入定位、地图、导航等相关的服务

Android 地图 SDK HOT 地图 包含地图显示、室内外一体化地图查看、兴趣点搜索、地理编码、离线地图等功能。 Android 定位 SDK HOT 搜索 包含获取定位结果、逆地理编码（地址文字描述）、以及地理围栏功能 Android 导航 SDK HOT 导航 包含路径规划、模拟导航、GPS 定位、自定义导航界面、获取导航播报信息等功能

iOS 平台

开发者可以轻松地在自己的 iOS 应用中加入定位、地图、导航等相关的服务

iOS 地图 SDK HOT 地图 包含地图显示与操作、兴趣点搜索、地理编码、路线规划等功能 iOS 定位 SDK HOT 搜索 包含单次定位、连续定位、逆地理信息、地理围栏等功能 iOS 导航 SDK HOT 导航 包含路径规划（含实时交通）、模拟导航、GPS 定位、自定义导航界面、获取导航播报信息等功能

地图工具

高德地图开放平台为开发者提供地图应用工具

坐标拾取器 搜索 根据关键字查询经纬度坐标，或根据经纬度反查该点所在位置 GeoHUB 定制地图 HOT 地图 数据中心与自定义地图、地图 JS API、SDK 等产品全面联通。提供地图样式自定义、数据上图等一站式服务，定制您的专属地图 地图快速生成器 地图 通过可视化操作，进行标注添加和地图编辑，最终生成可分享的地图链接和完整代码 地图名片 地图 将公司、个人的地址制作成地图名片，适合于展示型网站
