# PawLink 快速入门指南

欢迎使用 PawLink 宠物救助平台！本指南将帮助您在 5 分钟内快速上手。

---

## 📱 第一步：下载与安装

### iOS 用户
1. 打开 App Store
2. 搜索 "PawLink"
3. 点击"获取"并完成安装
4. 最低系统要求：iOS 14.0 或更高版本

### Android 用户
1. 打开 Google Play 商店
2. 搜索 "PawLink"
3. 点击"安装"
4. 最低系统要求：Android 9.0 或更高版本

### Web 用户
直接访问：[https://pawlink.app](https://pawlink.app)（示例网址）

---

## 🎯 第二步：完成实名认证

**为什么需要认证？**
实名认证是 PawLink 建立信任、保护动物安全的重要基础，所有用户必须完成认证才能使用核心功能。

### 认证步骤

1. **打开 APP，首次启动自动进入认证页面**

2. **填写基本信息**
   - 真实姓名
   - 身份证号码
   - 上传身份证正反面照片（支持 JPG/PNG，单张 < 5MB）

3. **人脸识别验证**
   - 按照提示进行人脸识别
   - 确保光线充足、面部清晰
   - 如识别失败，可重试 3 次

4. **等待审核**
   - 预计审核时间：24 小时内
   - 审核结果将通过 APP 通知和短信告知

5. **审核通过，开始使用！**

**小贴士**：
- 身份证照片请保持清晰、完整，四角可见
- 人脸识别时请摘下口罩、眼镜等遮挡物
- 认证信息仅用于平台安全管理，严格保密

## 🗺️ 第三步：探索核心功能

### 功能 1：查找附近的宠物（地图模式）

**使用场景**：我想领养一只宠物 / 我想帮助附近需要救助的动物

**操作步骤**：
1. 点击底部导航栏"首页"（地图图标）
2. 允许 APP 获取位置权限
3. 地图自动定位并显示附近的宠物信息点
4. 点击地图上的标记查看详情

**标记颜色说明**：
- 🟢 绿色：待领养宠物
- 🔴 红色：需紧急救助
- 🟡 黄色：医疗中
- ⚫ 灰色：已领养（历史记录）

**筛选功能**：
- 调整距离范围（1km / 5km / 10km / 不限）
- 选择动物类型（猫 / 狗 / 其他）
- 按状态筛选（待领养 / 需救助 / 紧急）

---

### 功能 2：发布救助信息

**使用场景**：我发现了一只流浪动物，需要帮助 / 我有宠物需要找领养人

**操作步骤**：
1. 点击底部导航栏中间的"+"按钮
2. 选择信息类型：
   - **待领养**：我的宠物需要找新家
   - **需救助**：发现流浪动物，需要帮助
   - **紧急救助**：动物受伤、生病，急需救助
3. 填写必填信息：
   - **上传照片**（至少 1 张，最多 9 张）
   - **选择位置**（GPS 定位或地图手动选点）
   - **动物类型**（猫 / 狗 / 其他）
   - **情况描述**（详细描述动物情况，如年龄、性别、健康状况）
   - **联系方式**（手机号必填，微信选填）
4. 点击"发布"
5. 信息审核通过后（通常 1-2 小时）将在地图上显示

**发布小贴士**：
- 照片清晰、完整，展示动物特征
- 详细描述有助于提高救助/领养成功率
- 紧急救助信息将优先推送给附近志愿者

---

### 功能 3：搜索与浏览宠物

**使用场景**：我想快速找到符合条件的宠物

**操作步骤**：
1. 点击底部导航栏"发现"（搜索图标）
2. 使用搜索过滤：
   - **距离筛选**：附近 3km / 5km / 10km / 不限
   - **动物类型**：猫 / 狗 / 其他
   - **状态筛选**：待领养 / 需救助 / 紧急
   - **高级筛选**：年龄、性别、是否已免疫等（Phase 2 功能）
3. 浏览搜索结果，点击查看详情
4. 收藏喜欢的宠物（点击爱心图标）

**智能推荐**：
- 系统会根据您的浏览历史和偏好推荐合适的宠物
- 新发布的信息会优先显示

---

### 功能 4：联系发布者

**使用场景**：我对某只宠物感兴趣，想了解更多信息

**操作步骤**：
1. 打开宠物详情页
2. 点击"联系 TA"按钮，选择联系方式：
   - **站内私信**：在 APP 内直接沟通（推荐）
   - **拨打电话**：直接拨打对方手机
   - **微信联系**：复制微信号添加好友
3. 发送消息，如：
   - "您好，我对这只猫很感兴趣，方便上门看望吗？"
   - "请问这只狗狗的健康状况如何？"

**沟通礼仪**：
- 礼貌用语，尊重对方
- 详细说明自己的情况（居住环境、养宠经验等）
- 按约定时间上门，如有变化及时告知
- 不要频繁骚扰

**安全提示**：
- 首次见面建议选择公共场所
- 避免单独前往偏僻地点
- 如遇到可疑情况，请使用"举报"功能

## 🆘 第四步：获取帮助

### 如何参与志愿者活动？
1. 点击"我的" → "成为志愿者"
2. 完成志愿者认证（技能认证 + 背景调查）
3. 浏览"志愿者活动"页面，认领任务
4. 完成任务后获得积分和成就

### 如何举报不良信息？
1. 打开可疑信息详情页
2. 点击右上角"..."
3. 选择"举报"，选择举报类型
4. 填写举报说明，提交

### 忘记密码 / 账号问题？
- 设备绑定登录，无需密码
- 如需更换设备，请联系客服

### 需要人工客服？
- APP 内：我的 → 帮助与反馈 → 联系客服
- 客服邮箱：support@pawlink.app（示例）
- 客服电话：400-XXX-XXXX（工作日 9:00-18:00）

## 🎁 第五步：了解进阶功能

### 众筹功能（Phase 2）
为需要医疗救助的动物发起众筹，获得爱心人士的支持。

**如何发起众筹**：
1. 进入已发布的救助信息详情
2. 点击"发起众筹"
3. 上传医疗费用单据
4. 设置众筹金额和截止时间
5. 等待审核通过

**如何捐赠**：
1. 浏览众筹列表
2. 选择想支持的项目
3. 选择捐赠金额（10 元 / 50 元 / 100 元 / 自定义）
4. 完成支付（微信 / 支付宝）
5. 获得电子捐赠证书

### 社区功能（Phase 2）
分享救助故事、领养心得、医疗知识，与其他爱心人士交流。

**发布帖子**：
1. 点击"社区" → "+"
2. 选择类型（救助故事 / 领养心得 / 医疗知识）
3. 填写标题、正文，上传图片/视频
4. 提交审核（预计 2 小时内）

**参与互动**：
- 点赞、评论、分享优质内容
- 关注感兴趣的用户
- 收藏有用的知识贴

### 积分与等级
通过参与平台活动获得积分，提升等级，解锁更多权益。

**积分获取方式**：
- 完成认证：+100 分
- 发布信息：+10 分
- 成功领养：+50 分
- 参与志愿者活动：+10-50 分
- 捐赠：+5-100 分（根据金额）

**等级权益**：
- LV1 新手：基础功能
- LV2 爱心使者：优先推荐
- LV3 救助达人：专属标识
- LV4 公益之星：优先审核、专属客服

## ⚠️ 重要提示

### 隐私与安全
- 您的实名信息仅用于平台认证，不会公开展示
- 联系方式仅在发起联系后双方可见
- 位置信息仅显示模糊位置（误差 500 米），详细地址需联系后可见

### 平台规则
- 禁止发布虚假信息（将被封号）
- 禁止宠物买卖、繁殖等商业行为
- 禁止虐待动物、极端言论
- 违规行为将扣除信用分，严重者永久封禁

### 动物保护理念
- 领养代替购买
- 不离不弃，对生命负责
- 尊重动物福利，反对虐待
- 科学养宠，定期体检免疫
