## 1. 账号与认证

### Q1.1 为什么必须实名认证？

**答**：实名认证是 PawLink 建立信任、保护动物安全的重要基础。

**原因**：
- 防止虚假信息泛滥（当前行业虚假信息占比达 30%+）
- 降低虐待动物风险
- 建立可追溯的责任体系
- 提高领养成功率（国外经验证明认证平台领养成功率提升 2-3 倍）

**隐私保护**：
- 您的实名信息仅用于平台认证，严格保密
- 对外展示时，姓名脱敏（如"张\*\*"），身份证号脱敏（如"110101\*\*\*\*\*\*\*\*1234"）
- 符合《个人信息保护法》要求

### Q1.2 人脸识别失败怎么办？

**常见失败原因**：
1. **光线问题**：光线太暗或逆光
   - 解决方法：在光线充足的环境下重试
2. **面部遮挡**：口罩、眼镜、帽子、刘海
   - 解决方法：摘下遮挡物重试
3. **姿势不正确**：角度偏斜、距离太近或太远
   - 解决方法：正面面对摄像头，保持 30-50cm 距离
4. **网络问题**：网络不稳定导致上传失败
   - 解决方法：切换到稳定的 WiFi 或 4G 网络

**仍然失败**：
- 系统提供 3 次重试机会
- 3 次失败后可申请人工审核通道
- 人工审核时间：1-2 个工作日

### Q1.3 忘记账号/密码怎么办？

**答**：PawLink 采用设备绑定登录，无需密码。

**说明**：
- 每个设备自动创建唯一账号
- 首次打开 APP 自动注册并绑定设备
- 无需记忆密码，打开即用

**更换设备**：
- 如需更换设备，请联系客服协助账号迁移
- 客服邮箱：support@pawlink.app
- 客服电话：400-XXX-XXXX

### Q1.4 志愿者认证需要多久？

**答**：通常 3-5 个工作日。

**审核内容**：
- 实名认证状态（必须已完成）
- 技能资质证明（如有上传）
- 个人经验描述
- 背景调查（无不良记录）

**加快审核**：
- 上传相关资格证书（如兽医证、驾驶证）
- 详细填写救助经验
- 保持联系方式畅通

### Q1.5 机构认证审核严格吗？

**答**：是的，机构认证审核较为严格。

**审核标准**：
- 营业执照/社会组织登记证书真实有效
- 实地验证通过（志愿者上门拍照）
- 场地、设施符合动物救助基本条件
- 负责人实名认证通过
- 无违规记录

**审核时间**：3-5 个工作日

**拒绝原因**：
- 证件信息不符
- 实地验证不通过
- 存在动物虐待或违规记录

## 2. 发布与搜索

### Q2.1 发布信息为什么审核不通过？

**常见拒绝原因**：
1. **照片不清晰**：图片模糊、光线太暗、动物特征不明显
   - 解决：重新拍摄清晰照片上传
2. **信息不完整**：必填项未填写或描述过于简单
   - 解决：完善动物详细信息、位置、联系方式
3. **疑似虚假信息**：同一账号短时间发布大量类似信息
   - 解决：确保信息真实，避免重复发布
4. **疑似商业行为**：包含买卖、繁殖等关键词
   - 解决：PawLink 仅支持救助和领养，不支持交易
5. **位置信息异常**：位置偏远或与照片不符
   - 解决：确保位置准确，GPS 定位或地图选点

**申诉方法**：
- 我的发布 → 选择被拒绝的信息 → "申诉"
- 说明情况，提供补充材料
- 1 个工作日内重新审核

### Q2.2 如何提高信息曝光率？

**方法一：完善信息**
- 上传 5-9 张高质量照片（不同角度）
- 详细描述动物情况（外观、性格、健康状况）
- 填写准确位置信息

**方法二：及时更新状态**
- 定期更新动物情况（如"已完成免疫"、"健康恢复良好"）
- 状态更新会重新推送给关注用户

**方法三：选择合适的类型**
- "紧急救助"优先级最高，优先推送
- 但请确保确实紧急，避免滥用

**方法四：分享到社交平台**
- 使用"分享"功能生成海报或链接
- 分享到微信、朋友圈、微博等
- 扩大传播范围

**方法五：提升账号等级**
- 高等级用户发布的信息优先推荐
- 通过完成认证、参与志愿者活动等提升等级

### Q2.3 搜索不到想要的宠物怎么办？

**检查筛选条件**：
- 距离范围是否太小？建议扩大到 10km 或不限
- 类型筛选是否太严格？尝试选择"全部"
- 状态筛选是否限制？尝试选择"全部"

**使用关键词搜索**：
- 发现 → 搜索框输入关键词
- 如"橘猫"、"金毛"、"人民广场附近"

**开启推荐功能**：
- 发现 → "为您推荐"
- 系统根据您的喜好智能推荐

**收藏关注**：
- 收藏感兴趣的宠物
- 关注发布者
- 有新发布会优先推送

### Q2.4 发布后多久会显示在地图上？

**答**：审核通过后 1-2 小时内显示。

**审核时间**：
- AI 初审：几分钟内完成
- 人工复审：1-2 小时（工作时间）
- 非工作时间：次日上午完成

**紧急救助优先**：
- 标记为"紧急救助"的信息优先审核
- 通常 30 分钟内完成

**查看审核状态**：
- 我的发布 → 查看信息状态
- 状态说明：
  - "审核中"：等待审核
  - "已发布"：审核通过，已上线
  - "已拒绝"：审核未通过，查看拒绝原因

### Q2.5 如何修改已发布的信息？

**步骤**：
1. 我的 → 我的发布
2. 选择要修改的信息
3. 点击"编辑"
4. 修改内容（照片、描述、位置、联系方式等）
5. 重新提交审核

**注意**：
- 修改后需要重新审核
- 审核期间信息仍然显示（显示修改前的内容）
- 审核通过后更新为新内容

**不可修改项**：
- 信息类型（待领养 / 需救助 / 紧急救助）
- 动物类型（猫 / 狗 / 其他）

## 3. 沟通与联系

### Q3.1 对方不回复我的消息怎么办？

**可能原因**：
1. **对方未开启消息通知**：可能没看到消息
2. **对方已找到领养人/救助者**：忘记更新状态
3. **对方忙碌**：暂时无法回复

**建议**：
- 等待 24-48 小时再次发送礼貌提醒
- 如果长时间无回复（3 天以上），可尝试拨打电话
- 同时浏览其他合适的宠物信息

**避免骚扰**：
- 不要短时间内频繁发送消息（系统限制：1 分钟内最多 10 条）
- 不要发送无意义的消息

### Q3.2 如何判断对方是否可信？

**查看认证状态**：
- ✅ **已实名认证**：基本可信
- ✅ **志愿者认证**：活跃救助者，可信度高
- ✅ **机构认证**：正规机构，可信度最高

**查看历史记录**：
- 点击对方头像 → 查看主页
- 查看发布记录（数量、质量）
- 查看用户等级（高等级说明长期活跃）

**查看评价**：
- 对方的志愿者评价（如有）
- 对方的社区帖子（了解其救助理念）

**警惕信号**：
- 要求提前支付费用（PawLink 领养免费，仅可能有绝育、免疫费用）
- 联系方式频繁变更
- 发布大量类似信息
- 拒绝上门看望或视频通话

**安全建议**：
- 首次见面选择公共场所
- 可邀请朋友陪同
- 保留沟通记录（截图）
- 如遇可疑情况，使用"举报"功能

### Q3.3 举报后会怎样？

**举报流程**：
1. 聊天界面/信息详情页 → 右上角"..." → "举报"
2. 选择举报类型（虚假信息 / 骚扰 / 欺诈 / 虐待动物 / 其他）
3. 上传证据（聊天截图、照片等）
4. 提交举报

**平台处理**：
- **24 小时内**：人工审核举报内容
- **核实后**：
  - 轻微违规：警告、限制功能
  - 严重违规：扣除信用分、下架信息
  - 恶意违规：封禁账号（7 天 / 30 天 / 永久）

**举报反馈**：
- 处理结果通过站内信通知举报人
- 保护举报人隐私（被举报人不知道谁举报）

**恶意举报**：
- 系统检测恶意举报行为（重复举报、虚假举报）
- 恶意举报者将被反向处罚

### Q3.4 可以在站外交易吗？

**答**：不建议，存在风险。

**平台规则**：
- PawLink 提倡"领养代替购买"
- 禁止宠物买卖、繁殖等商业行为
- 发现交易行为将封禁账号

**为什么不建议站外交易**：
1. **无法追溯**：出现纠纷平台无法介入
2. **容易被骗**：无法验证对方身份
3. **动物安全**：无法保障动物去向

**合理费用**：
- 领养本身免费
- 可能产生的费用：
  - 绝育费用：300-800 元
  - 免疫费用：100-300 元
  - 体检费用：200-500 元
- 这些费用应由双方协商，凭单据支付

## 4. 志愿者相关

### Q4.1 如何成为志愿者？

**步骤**：
1. 完成实名认证
2. 我的 → 成为志愿者 → 志愿者认证
3. 填写技能类型、个人经验
4. 上传证明材料（选填）
5. 授权背景调查
6. 等待审核（3-5 个工作日）

**技能要求**：
- 无强制要求，有爱心即可
- 常见技能：救助、运输、寄养、医疗、宣传
- 有相关经验或资质更佳

### Q4.2 志愿者任务有报酬吗？

**答**：志愿者活动以公益为主，无现金报酬，但有激励机制。

**激励方式**：
- **积分奖励**：完成任务获得 10-50 积分
- **虚拟徽章**：完成特定成就解锁徽章
- **等级提升**：积分累计提升志愿者等级
- **公益证书**：年度贡献证书（可用于升学、就业）
- **优惠权益**：合作商家优惠券（如宠物医院折扣）

**部分任务有补贴**：
- 运输协助：油费补贴（双方协商）
- 临时寄养：寄养补贴（双方协商）
- 医疗陪护：可能有交通补贴

### Q4.3 认领任务后能取消吗？

**答**：可以，但建议谨慎认领。

**取消方法**：
- 我的 → 志愿者中心 → 我的任务
- 选择任务 → "取消参加"
- 填写取消原因（选填）
- 确认取消

**取消影响**：
- **任务开始 24 小时前取消**：无影响
- **任务开始 24 小时内取消**：扣除 5 积分
- **已签到后取消**：扣除 10 积分，影响信用评分
- **频繁取消**（1 个月内 3 次以上）：限制认领任务 30 天

**建议**：
- 认领前确认时间安排
- 如有变化尽早取消
- 紧急情况及时联系发布者沟通

### Q4.4 志愿者等级有什么用？

**等级划分**：
- LV1 志愿者：0-100 分
- LV2 活跃志愿者：101-500 分
- LV3 资深志愿者：501-1500 分
- LV4 金牌志愿者：1501-5000 分
- LV5 公益之星：5000 分以上

**等级权益**：

| 等级 | 权益 |
|------|------|
| LV1 | 基础功能，可认领任务 |
| LV2 | 优先获得任务推送 |
| LV3 | 专属徽章、优先审核 |
| LV4 | 年度公益证书、优先客服 |
| LV5 | 平台代言人、专属活动、特殊奖励 |

## 5. 众筹与捐赠

### Q5.1 众筹的钱真的会用在动物身上吗？

**答**：是的，平台有严格的资金监管机制。

**资金管理**：
1. **第三方托管**：所有众筹资金由第三方支付平台托管
2. **冻结机制**：资金冻结，发起人无法直接提现
3. **凭证审核**：
   - 发起人需上传医疗单据（诊断书、费用清单）
   - 众筹成功后需上传资金使用凭证（发票、收据）
4. **透明公示**：资金使用情况在众筹页面公开
5. **平台抽成**：平台收取 5% 手续费（用于平台运营和监管成本）

**监督机制**：
- 捐赠者可查看资金使用明细
- 发现问题可举报
- 平台定期抽查审计

**众筹失败退款**：
- 未达到目标金额，自动退回捐赠者
- 3 个工作日内原路退回
- 平台不收取任何费用

### Q5.2 捐赠后能获得什么？

**物质激励**：
- **电子捐赠证书**：可下载、分享
- **公益值**：捐赠金额 = 公益值（如捐 50 元 = 50 公益值）
- **虚拟徽章**：
  - 爱心新人：首次捐赠
  - 爱心使者：累计捐赠 ≥ 100 元
  - 爱心天使：单笔捐赠 ≥ 500 元
  - 公益之星：累计捐赠 ≥ 1000 元

**精神激励**：
- 个人主页展示捐赠徽章
- 年度捐赠排行榜（可选择匿名）
- 众筹成功后收到感谢消息
- 看到动物康复、被领养的后续反馈

**社会价值**：
- 帮助真正需要救助的动物
- 传递爱心，影响更多人

### Q5.3 可以申请退款吗？

**众筹捐赠退款**：

**可以退款的情况**：
- 众筹失败（未达到目标金额）：自动退款
- 众筹项目被平台认定为虚假信息：强制退款
- 支付错误（重复支付）：联系客服退款

**不可退款的情况**：
- 众筹成功后改变主意
- 对发起人使用资金方式不满意（但可举报）

**退款流程**：
- 联系客服：support@pawlink.app
- 说明退款原因
- 提供支付凭证
- 3-5 个工作日内处理

**平台捐赠退款**：
- 捐赠成功后不支持退款
- 捐赠前请谨慎确认

## 6. 安全与隐私

### Q6.1 我的个人信息会泄露吗？

**答**：不会，平台严格保护用户隐私。

**隐私保护措施**：

1. **实名信息脱敏**：
   - 姓名：张\*\*
   - 身份证号：110101\*\*\*\*\*\*\*\*1234

2. **联系方式隐藏**：
   - 手机号、微信号仅在发起联系后双方可见
   - 未联系前不可见

3. **位置信息模糊化**：
   - 公开展示：模糊位置（误差 500 米）
   - 详细地址：仅在发起联系后可见

4. **数据加密**：
   - HTTPS 加密传输
   - 敏感数据本地加密存储

5. **权限控制**：
   - 严格的数据访问权限控制
   - 管理员查看需留存审计日志

**合规认证**：
- 符合《个人信息保护法》
- 符合《网络安全法》
- 定期安全审计

### Q6.2 如何防止被骗？

**识别骗局信号**：
1. **要求提前付款**：
   - 正常领养免费（仅可能有绝育、免疫费用）
   - 骗子会以各种理由要求转账（押金、运费等）
2. **拒绝见面**：
   - 骗子常以距离远、忙碌等理由拒绝上门看望
   - 建议坚持见面或视频通话
3. **信息可疑**：
   - 照片模糊或从网上盗图
   - 描述前后不一致
   - 联系方式频繁变更
4. **异常行为**：
   - 短时间发布大量类似信息
   - 同一手机号注册多个账号

**防骗建议**：
1. **优先选择认证用户**：
   - 实名认证 ✅
   - 志愿者认证 ✅
   - 机构认证 ✅（可信度最高）
2. **坚持站内沟通**：
   - 保留聊天记录
   - 出现纠纷可举报
3. **见面验证**：
   - 坚持上门看望或要求视频通话
   - 首次见面选择公共场所
4. **凭单据付费**：
   - 如需支付绝育、免疫费用，要求提供医院单据
   - 避免直接转账给个人
5. **保持警惕**：
   - 遇到可疑情况立即停止沟通
   - 使用"举报"功能

**被骗后处理**：
1. 保留证据（聊天记录、转账记录）
2. 立即举报（平台会封禁骗子账号）
3. 报警（涉及金额较大时）

### Q6.3 如何设置隐私保护？

**隐私设置路径**：我的 → 设置 → 隐私设置

**可设置项**：

1. **位置隐私**：
   - 公开精确位置：关闭（默认）
   - 允许附近的人看到我：开启/关闭

2. **联系方式隐私**：
   - 手机号：仅联系后可见（不可修改）
   - 微信号：仅联系后可见（不可修改）

3. **个人资料隐私**：
   - 性别：公开 / 不公开
   - 生日：公开 / 不公开
   - 所在城市：公开 / 不公开

4. **浏览隐私**（Phase 2）：
   - 匿名浏览：开启后浏览不留痕迹

**推荐设置**：
- 关闭"公开精确位置"（保护家庭住址）
- 根据个人喜好设置个人资料公开范围

## 7. 技术问题

### Q7.1 APP 闪退怎么办？

**解决方法**：

1. **重启 APP**：
   - 完全退出 APP
   - 重新打开

2. **清除缓存**：
   - 设置 → 存储空间 → 清除缓存
   - 不影响账号数据

3. **检查版本**：
   - 设置 → 关于 → 查看版本
   - 如有新版本，建议更新

4. **重启手机**：
   - 重启设备后再次尝试

5. **重装 APP**：
   - 卸载 APP
   - 重新下载安装
   - 数据会自动同步恢复

**仍然无法解决**：
- 联系客服反馈问题
- 提供设备型号、系统版本、复现步骤

### Q7.2 定位不准确怎么办？

**检查权限**：
1. iOS：设置 → 隐私 → 定位服务 → PawLink → 始终允许
2. Android：设置 → 应用 → PawLink → 权限 → 位置 → 始终允许

**提高定位精度**：
1. **开启 GPS**：确保 GPS 功能开启
2. **切换定位模式**：
   - Android：设置 → 位置 → 模式 → 高精度
3. **室外使用**：室内 GPS 信号弱，建议室外重新定位
4. **网络辅助**：切换到 WiFi 或 4G 网络

**手动选点**：
- 如 GPS 定位不准，可使用"地图选点"手动标记位置

### Q7.3 照片上传失败怎么办？

**常见原因及解决方法**：

1. **文件过大**：
   - 限制：单张 < 10MB
   - 解决：使用 APP 内置裁剪工具压缩照片

2. **格式不支持**：
   - 支持格式：JPG、PNG
   - 解决：转换为支持的格式

3. **网络问题**：
   - 检查网络连接
   - 切换到稳定的 WiFi 或 4G

4. **相册权限**：
   - iOS：设置 → 隐私 → 照片 → PawLink → 所有照片
   - Android：设置 → 应用 → PawLink → 权限 → 存储 → 允许

5. **存储空间不足**：
   - 检查手机存储空间
   - 清理不必要的文件

### Q7.4 收不到通知怎么办？

**检查通知权限**：
1. **iOS**：
   - 设置 → 通知 → PawLink
   - 允许通知：开启
   - 显示在锁定屏幕：开启
2. **Android**：
   - 设置 → 应用 → PawLink → 通知
   - 允许通知：开启

**检查 APP 内设置**：
- 我的 → 设置 → 消息通知设置
- 确保相应通知类型已开启

**检查免打扰时段**：
- 设置 → 消息通知设置 → 免打扰时段
- 确认当前时间不在免打扰时段内

**检查网络**：
- 通知需要网络连接
- 确保设备联网

**仍然无法解决**：
- 重启 APP
- 重装 APP
- 联系客服

### Q7.5 聊天记录丢失了怎么办？

**聊天记录备份机制**：
- 本地保存：最近 30 天聊天记录
- 云端备份：最近 30 天聊天记录

**恢复聊天记录**：
1. **未卸载 APP**：
   - 聊天记录本地保存
   - 重启 APP 即可恢复
2. **已卸载 APP**：
   - 重新安装后自动从云端恢复
   - 恢复最近 30 天聊天记录
3. **更换设备**：
   - 联系客服协助账号迁移
   - 迁移后可恢复聊天记录

**无法恢复的情况**：
- 超过 30 天的聊天记录
- 账号注销后

**建议**：
- 重要信息及时截图保存
- 定期备份重要聊天记录

## 8. 其他问题

### Q8.1 领养宠物需要什么条件？

**PawLink 平台无硬性条件限制**，但建议具备以下条件：

**基本条件**：
1. **稳定的居住环境**：
   - 有固定住所（租房需房东同意养宠）
   - 空间适合宠物活动
2. **经济能力**：
   - 能够承担宠物日常开销（食物、医疗、用品）
   - 预算：猫约 200-500 元/月，狗约 300-800 元/月
3. **时间精力**：
   - 有时间陪伴和照顾宠物
   - 如经常出差，需有人代为照料
4. **家人同意**：
   - 全家人都同意并愿意接受宠物

**加分条件**：
- 有养宠经验
- 了解宠物习性和基本护理知识
- 愿意定期带宠物体检、免疫

**发布者可能的附加要求**：
- 定期回访（提供照片/视频）
- 签订领养协议
- 承诺不离不弃

**不建议领养的情况**：
- 一时冲动，未深思熟虑
- 仅为满足孩子短暂兴趣
- 居住环境不允许（如严格禁宠小区）
- 家人反对或有人对宠物过敏

### Q8.2 领养是完全免费的吗？

**领养本身免费**，但可能产生以下合理费用：

**可能的费用**：
1. **绝育费用**：300-800 元
   - 如发布者已为宠物绝育，可能要求领养人承担
2. **免疫费用**：100-300 元
   - 如宠物已完成疫苗接种，可能要求分担费用
3. **体检费用**：200-500 元
   - 部分发布者会要求领养前体检
4. **运输费用**：视距离而定
   - 跨城市领养可能产生运输费用

**费用原则**：
- 双方协商一致
- 凭单据支付（医院发票、收据）
- 避免直接转账给个人

**警惕**：
- 如发布者要求不合理高额费用（如"押金"数千元）
- 可能是骗局，请谨慎或举报

### Q8.3 如何提高领养成功率？

**对于领养者**：

1. **完善个人资料**：
   - 完成实名认证
   - 填写详细个人信息
   - 提升用户等级

2. **主动沟通**：
   - 礼貌、详细说明自己的情况
   - 展示养宠经验和诚意
   - 提供居住环境照片

3. **灵活调整条件**：
   - 不要过于挑剔品种、年龄
   - 考虑领养成年猫/狗（竞争少）
   - 扩大搜索范围

4. **耐心等待**：
   - 领养需要双向匹配
   - 多浏览、多沟通、多上门

**对于发布者**：

1. **优质信息**：
   - 高清照片（5-9 张）
   - 详细描述动物情况
   - 及时更新状态

2. **合理条件**：
   - 不要设置过高门槛（如"必须有房"）
   - 重点关注领养者的爱心和责任心

3. **积极回复**：
   - 及时回复咨询
   - 耐心解答问题
   - 安排上门看望

4. **后续回访**：
   - 建立长期联系
   - 定期了解宠物情况
   - 分享成功案例激励他人

### Q8.4 可以同时联系多个发布者吗？

**答**：可以，但建议有选择性地联系。

**建议做法**：
1. **筛选目标**：
   - 先浏览、收藏感兴趣的宠物
   - 精选 3-5 个最心仪的
2. **逐个沟通**：
   - 避免同时联系太多（容易管理混乱）
   - 先联系最心仪的，如不合适再联系其他
3. **及时反馈**：
   - 如决定领养某只宠物，及时告知其他发布者
   - 避免浪费彼此时间

**不建议做法**：
- 同时联系 10+ 个发布者（容易被认为不诚心）
- 联系后长期不回复
- 约定上门后爽约

### Q8.5 宠物生病了平台会负责吗？

**答**：平台不承担宠物医疗责任。

**原因**：
- PawLink 是信息平台，连接救助者和领养者
- 宠物健康由发布者和领养者协商确定
- 平台无法验证每只宠物的健康状况

**建议**：
1. **领养前**：
   - 要求查看体检报告
   - 上门查看宠物健康状况
   - 建议领养前带宠物体检
   - 与发布者明确健康责任（如"7 天内出现问题可退回"）
2. **领养后**：
   - 定期带宠物体检、免疫
   - 购买宠物保险（可选）
   - 遇到健康问题及时就医

**特殊情况**：
- 如发布者故意隐瞒宠物重大疾病
- 可向平台举报
- 平台将封禁其账号
