## 1. 账号与认证

### 1.1 用户注册

**注册方式**：
PawLink 采用基于设备的自动注册，首次打开 APP 自动创建账号。

**初始化流程**：
1. 下载并打开 APP
2. 阅读并同意《用户协议》和《隐私政策》
3. 允许必要权限（位置、相机、相册、通知）
4. 自动创建设备绑定账号

**权限说明**：
- **位置权限**（必需）：用于地图定位、附近宠物搜索
- **相机权限**（必需）：用于拍照上传、人脸识别认证
- **相册权限**（必需）：用于选择照片上传
- **通知权限**（推荐）：接收重要消息和救助提醒

### 1.2 实名认证

**认证入口**：首页 → 顶部提示 "请完成实名认证" → 点击进入

**认证步骤**：

#### 步骤 1：填写个人信息
- **真实姓名**：请使用与身份证一致的姓名
- **身份证号**：支持 18 位身份证号，系统自动校验格式
- 点击"下一步"

#### 步骤 2：上传身份证照片
- **正面照片**：国徽面，需拍摄清晰、完整、四角可见
- **反面照片**：人像面，需拍摄清晰、完整、四角可见
- 支持格式：JPG、PNG
- 单张大小限制：< 5MB
- 系统自动 OCR 识别，核验信息一致性
- 点击"下一步"

#### 步骤 3：人脸识别验证
- 按照屏幕提示进行人脸识别
- **注意事项**：
  - 光线充足，避免逆光或暗光
  - 面部清晰，无遮挡（摘下口罩、眼镜、帽子）
  - 正面面对摄像头
  - 按提示完成动作（如眨眼、张嘴、转头）
- 支持重试：失败后最多可重试 3 次
- 3 次失败后提供人工审核通道

#### 步骤 4：等待审核
- **审核时间**：通常 24 小时内完成
- **审核结果**：通过 APP 通知和短信告知
- **审核通过**：自动开通全部功能
- **审核失败**：提示失败原因，可重新提交

**认证状态**：
- **未认证**：仅可浏览，无法发布信息、联系他人
- **审核中**：等待人工审核
- **已认证**：完整功能访问权限

**隐私保护**：
- 认证信息仅用于平台安全管理
- 身份证号不对外展示（脱敏为 110101********1234）
- 实名信息不公开，仅管理员可查看

### 1.3 志愿者认证

**认证入口**：我的 → 成为志愿者 → 志愿者认证

**前置条件**：已完成实名认证

**认证步骤**：

1. **选择技能类型**（至少 1 项，最多 3 项）：
   - **救助技能**：能够现场救助流浪动物
   - **医疗基础**：具备基本医疗知识或兽医资格
   - **运输协助**：有车，可协助运输动物
   - **临时寄养**：可提供临时寄养场所
   - **其他技能**：如摄影、文案、宣传等

2. **上传证明材料**（选填）：
   - 相关资格证书（如兽医资格证）
   - 驾驶证（运输协助）
   - 住所照片（临时寄养）

3. **填写个人经验描述**（必填，50-500 字）：
   - 救助经验
   - 技能特长
   - 可投入时间

4. **授权背景调查**：
   - 阅读并同意《志愿者背景调查授权书》
   - 授权平台进行必要的背景调查

5. **提交审核**：
   - 审核时间：3-5 个工作日
   - 审核通过后即可认领志愿者任务

**志愿者权益**：
- 优先获得志愿者任务推送
- 参与任务获得积分奖励
- 解锁志愿者专属徽章
- 累计贡献可获得公益证书

### 1.4 救助机构认证

**认证入口**：我的 → 机构认证 → 填写机构信息

**前置条件**：管理员已完成实名认证

**认证步骤**：

1. **选择机构类型**：
   - **正规救助机构**：有营业执照或社会组织登记证书
   - **个人救助站**：个人运营的救助基地

2. **上传资质证明**：
   - 营业执照 / 社会组织登记证书（正规机构必填）
   - OCR 自动识别机构信息

3. **填写机构信息**：
   - **机构名称**：与证书一致
   - **机构地址**：实际运营地址（支持地图选点）
   - **联系方式**：机构电话、负责人手机
   - **救助规模**：场地大小、动物数量、年救助量

4. **提供实地验证地址**：
   - 平台将安排志愿者实地验证
   - 拍摄现场照片上传

5. **提交审核**：
   - 审核时间：3-5 个工作日
   - 审核通过后可享受机构特权

**机构特权**：
- 机构专属认证标识
- 批量发布宠物信息
- 数据统计报表
- 优先推荐曝光
- 接入众筹功能

## 2. 地图与导航

### 2.1 首页地图

**进入方式**：底部导航栏 → 首页（地图图标）

**地图功能**：

#### 自动定位
- 首次使用需授权位置权限
- 地图自动定位当前位置（蓝色定位点）
- 定位精度：通常 < 50 米（GPS 信号良好时）

#### 宠物信息点标记
地图上显示附近的宠物信息，不同颜色标记不同状态：

| 颜色 | 状态 | 说明 |
|------|------|------|
| 🟢 绿色 | 待领养 | 健康宠物，寻找新家 |
| 🔴 红色 | 紧急救助 | 动物受伤/生病，急需救助 |
| 🟠 橙色 | 需救助 | 流浪动物，需要帮助 |
| 🟡 黄色 | 医疗中 | 正在接受治疗 |
| ⚫ 灰色 | 已领养/已救助 | 历史记录，可隐藏 |

#### 点击标记查看详情
- 点击地图上的标记点
- 弹出简略信息卡片
- 点击卡片进入详情页

#### 热力图显示（可选）
- 右上角切换"热力图模式"
- 显示救助密度高的区域（红色深浅表示密度）

### 2.2 地图筛选

**筛选入口**：地图页面顶部筛选栏

**筛选选项**：

#### 距离筛选
- **1km 内**：步行可达
- **5km 内**（默认）：骑行/短途驾车可达
- **10km 内**：驾车可达
- **50km 内**：跨区域
- **不限距离**：全部信息

#### 动物类型筛选
- **全部**（默认）
- **猫**
- **狗**
- **其他**（兔子、鸟类、仓鼠等）

#### 状态筛选
- **全部**（默认）
- **待领养**
- **需救助**
- **紧急救助**
- **医疗中**

#### 时间筛选
- **全部**（默认）
- **今日发布**
- **本周发布**
- **本月发布**

**筛选组合**：
- 支持多条件组合筛选（AND 逻辑）
- 实时更新地图显示结果
- 显示筛选结果数量（如"附近 3km 内共 12 只待领养猫"）

**清除筛选**：
- 点击"重置"按钮清除所有筛选条件

### 2.3 地图操作

**缩放**：
- 双指捏合/分开：缩放地图
- 双击：放大一级
- 缩放时标记聚合显示（避免重叠）

**移动**：
- 单指拖动：移动地图
- 移动后显示"重新搜索此区域"按钮

**回到当前位置**：
- 点击右下角"定位"按钮
- 地图自动回到当前位置

**切换地图模式**：
- 标准地图：默认模式
- 卫星地图：真实卫星图像
- 路况地图：显示实时路况（仅大城市）

## 3. 信息发布与管理

### 3.1 发布救助信息

**发布入口**：底部导航栏 → "+"按钮 → 发布信息

**发布流程**：

#### 步骤 1：选择信息类型

**待领养**：
- 适用场景：我的宠物需要找新家
- 要求：宠物健康状况良好，已完成基础免疫

**需救助**：
- 适用场景：发现流浪动物，需要帮助
- 要求：动物暂无生命危险，但需要救助

**紧急救助**：
- 适用场景：动物受伤、生病、被困，急需救助
- 特点：优先推送给附近志愿者，24/7 响应

#### 步骤 2：上传照片

**上传要求**：
- **数量**：至少 1 张，最多 9 张
- **格式**：JPG、PNG
- **大小**：单张 < 10MB
- **内容**：清晰展示动物特征、全貌

**拍摄建议**：
- 正面照：清晰显示面部特征
- 侧面照：展示体型、毛色
- 细节照：特殊标记（如项圈、伤口）
- 环境照：发现地点周边环境

**照片编辑**：
- 支持裁剪、旋转
- 自动压缩优化（保持清晰度）
- 可调整照片顺序（首张为封面）

#### 步骤 3：选择位置

**GPS 定位**（推荐）：
- 点击"使用当前位置"
- 系统自动获取 GPS 坐标
- 显示详细地址

**地图选点**：
- 点击"地图选点"
- 在地图上点击精确位置
- 支持搜索地址定位

**隐私保护**：
- 公开展示：模糊位置（误差 500 米）
- 详细地址：仅在发起联系后双方可见

#### 步骤 4：填写动物信息

**基础信息**（必填）：
- **动物类型**：猫 / 狗 / 其他
- **性别**：公 / 母 / 未知
- **年龄**：幼年（< 1 岁）/ 成年（1-7 岁）/ 老年（> 7 岁）/ 未知
- **体型**：小型 / 中型 / 大型（仅限狗）

**详细描述**（必填，50-1000 字）：
- 发现时间、地点
- 动物外观特征（毛色、体型、特殊标记）
- 健康状况（是否受伤、生病）
- 性格特点（亲人、胆小、活泼等）
- 其他重要信息

**健康信息**（选填）：
- 是否已免疫
- 是否已绝育
- 是否有慢性病
- 最近一次体检时间

#### 步骤 5：填写联系方式

**手机号**（必填）：
- 用于接收救助/领养咨询
- 仅在对方发起联系后可见

**微信号**（选填）：
- 方便快速沟通
- 仅在对方发起联系后可见

**备注说明**（选填）：
- 如"每天 18:00-22:00 方便接听电话"

#### 步骤 6：提交发布

- 点击"发布"按钮
- 系统自动审核（AI 初审）
- 审核通过：1-2 小时内显示在地图上
- 审核失败：提示失败原因（如照片不清晰、信息不完整）

**AI 审核内容**：
- 照片合规性检查（无违规内容）
- 信息完整性检查（必填项齐全）
- 重复信息检测（是否与已发布信息重复）
- 虚假信息识别（异常行为检测）

### 3.2 管理已发布信息

**查看已发布**：我的 → 我的发布

**信息状态**：
- **审核中**：等待审核
- **已发布**：正常展示中
- **已下架**：手动下架或已领养
- **已拒绝**：审核未通过

**可执行操作**：

#### 更新状态
- 点击信息 → "更新状态"
- 可选状态：
  - 待领养 → 医疗中 → 已领养
  - 需救助 → 医疗中 → 待领养
  - 紧急救助 → 医疗中 → 已救助
- 填写更新说明（如"已完成免疫接种"）
- 关注该信息的用户会收到更新通知

#### 编辑信息
- 点击信息 → "编辑"
- 可修改照片、描述、联系方式
- 重新提交审核

#### 下架信息
- 点击信息 → "下架"
- 适用场景：
  - 宠物已找到领养人
  - 动物已被救助
  - 信息填写错误
- 下架后信息不再显示在地图上
- 可在"已下架"列表中重新上架

#### 删除信息
- 点击信息 → "删除"
- 确认后永久删除（不可恢复）
- 建议先下架，确认无需后再删除

#### 分享信息
- 点击信息 → "分享"
- 生成分享海报或链接
- 分享到微信、朋友圈、微博等平台
- 扩大救助/领养传播范围

### 3.3 信息自动管理

**自动下架规则**：
- 状态更新为"已领养/已救助"后，7 天自动下架
- 长期未更新（90 天）的信息自动下架，提醒用户确认

**自动归档**：
- 下架 30 天后自动归档
- 归档信息不再显示，但可在历史记录中查看

**统计数据**：
- 浏览量：多少人查看过信息
- 收藏量：多少人收藏了信息
- 联系量：多少人发起联系

## 4. 搜索与筛选

### 4.1 宠物搜索

**搜索入口**：底部导航栏 → 发现（搜索图标）

**搜索方式**：

#### 关键词搜索
- 搜索框输入关键词
- 支持搜索：
  - 品种（如"加菲猫"、"金毛"）
  - 颜色（如"橘猫"、"黑猫"、"白狗"）
  - 地点（如"浦东新区"、"人民广场"）
- 搜索结果按相关度排序

#### 高级筛选

**基础筛选**：
- **距离**：1km / 5km / 10km / 50km / 不限
- **类型**：猫 / 狗 / 其他
- **状态**：待领养 / 需救助 / 紧急
- **性别**：公 / 母 / 不限
- **年龄**：幼年 / 成年 / 老年 / 不限

**高级筛选**（Phase 2）：
- **品种**：常见品种下拉选择
- **体型**：小型 / 中型 / 大型（仅限狗）
- **健康状况**：
  - ✅ 已免疫
  - ✅ 已绝育
  - ✅ 健康
- **性格特点**：亲人 / 活泼 / 安静 / 胆小
- **发布者类型**：个人 / 救助机构

**排序方式**：
- **距离优先**（默认）：由近到远
- **时间优先**：最新发布
- **匹配度优先**：根据用户偏好智能排序

### 4.2 智能推荐（Phase 2）

**推荐算法基于**：
- 浏览历史（您最近查看的宠物类型）
- 收藏记录（您收藏的宠物特征）
- 距离因素（附近优先推荐）
- 成功匹配预测（系统预测您的领养成功率）

**推荐入口**：
- 发现页面 → "为您推荐"标签
- 首页 → "猜你喜欢"卡片

**推荐标签**：
- **强烈推荐**：匹配度 > 80%
- **根据您的喜好推荐**：浏览历史相似
- **附近新发布**：距离近 + 时间新

### 4.3 收藏与历史

**收藏宠物**：
- 宠物详情页 → 点击"爱心"图标
- 收藏后可在"我的收藏"中快速查看
- 收藏的宠物状态更新时会收到通知

**查看收藏**：我的 → 我的收藏

**浏览历史**：
- 我的 → 浏览历史
- 自动记录最近 100 条浏览记录
- 可清空历史记录

## 5. 沟通与联系

### 5.1 站内私信

**发起私信**：
- 宠物详情页 → "联系 TA"→ "站内私信"
- 自动创建聊天会话

**聊天功能**：

#### 发送消息
- **文字消息**：
  - 输入框输入文字，点击发送
  - 支持表情符号
  - 敏感词自动过滤
- **图片消息**：
  - 点击"相册"图标
  - 选择照片（最多 9 张）
  - 单张 < 10MB

#### 快捷回复（预设模板）
- "您好，我对这只宠物很感兴趣"
- "方便上门看望吗？"
- "请问宠物的健康状况如何？"
- "我的居住环境：XXX"

#### 消息状态
- **发送中**：消息正在发送
- **已送达**：对方已收到
- **已读**：对方已阅读

#### 消息提醒
- 新消息通知（红点提示）
- 消息推送（可在设置中关闭）

**聊天记录**：
- 本地保存最近 30 天聊天记录
- 云端备份，重装应用可恢复
- 可搜索聊天内容

### 5.2 快速联系

**拨打电话**：
- 宠物详情页 → "联系 TA" → "拨打电话"
- 直接调起系统拨号界面
- 记录联系行为（用于统计）

**微信联系**：
- 宠物详情页 → "联系 TA" → "微信联系"
- 显示对方微信号
- 支持一键复制

**联系记录**：
- 我的 → 联系记录
- 查看所有发起/接收的联系请求

### 5.3 消息防骚扰

**频率限制**：

**拉黑功能**：
- 聊天界面 → 右上角"..." → "拉黑"

**免打扰**：
- 聊天界面 → 右上角"..." → "消息免打扰"
- 免打扰后不再收到该会话的推送通知

**举报功能**：
- 聊天界面 → 右上角"..." → "举报"

  - 虚假信息
  - 骚扰
  - 欺诈
  - 虐待动物
  - 其他
- 上传聊天截图作为证据
- 提交后平台将在 24 小时内处理

## 7. 众筹与捐赠

### 7.1 浏览众筹项目

**众筹列表入口**：首页 → 爱心众筹

**项目信息**：
- **封面照片**：动物照片或医疗单据
- **项目标题**：简要说明众筹目的
- **目标金额**：需要筹集的总金额（100-20,000 元）
- **已筹金额**：实时更新
- **进度条**：直观显示完成百分比
- **支持人数**：参与捐赠的人数
- **剩余时间**：距离截止时间（倒计时）

**项目状态**：
- **进行中**：正在筹款
- **已成功**：达到目标金额
- **已失败**：未达到目标，截止时间到
- **已结束**：资金已使用完毕

**筛选项目**：
- 按进度排序（完成度低优先）
- 按时间排序（即将截止优先）
- 按距离排序（附近优先）

### 7.2 发起众筹

**众筹入口**：我的发布 → 选择已发布信息 → "发起众筹"

**前置条件**：
- 已完成实名认证
- 已发布"需救助"或"紧急救助"信息
- 动物确实需要医疗救助

**发起流程**：

#### 步骤 1：上传医疗单据
- **单据照片**（必填，至少 1 张）：
  - 医院诊断书
  - 治疗方案
  - 费用清单
- OCR 自动识别金额和诊断信息

#### 步骤 2：填写众筹信息
- **众筹标题**（必填，10-30 字）：
  - 如"救助受伤流浪狗，急需手术费"
- **目标金额**（必填，100-20,000 元）：
  - 建议与医疗单据金额一致
- **资金用途说明**（必填，50-500 字）：
  - 详细说明资金使用计划
  - 如手术费、药费、住院费等明细
- **众筹截止时间**（必填，3-30 天）：
  - 建议根据治疗紧急程度设定

#### 步骤 3：提交审核
- 平台人工审核（1 个工作日内）
- 审核标准：
  - 医疗单据真实有效
  - 金额合理（不高于单据金额 20%）
  - 资金用途明确
- 审核通过后立即上线

### 7.3 支持众筹（捐赠）

**捐赠流程**：
1. 浏览众筹列表，选择想支持的项目
2. 点击"支持"按钮
3. 选择捐赠金额：
   - **预设金额**：10 元 / 50 元 / 100 元
   - **自定义金额**：最低 1 元，最高 5000 元
4. 选择支付方式：
   - 微信支付
   - 支付宝
5. 完成支付
6. 捐赠成功

**捐赠凭证**：
- 电子捐赠证书（可下载 / 分享）
- 捐赠记录显示在"我的捐赠"
- 年度捐赠总额统计

**捐赠激励**：
- 捐赠金额计入个人公益值
- 公益值可兑换虚拟徽章
- 大额捐赠者（单笔 ≥ 500 元）获得"爱心天使"徽章

### 7.4 众筹资金管理

**资金托管**：
- 所有众筹资金由第三方支付平台托管
- 资金冻结，发起人无法直接提现

**众筹成功**：
- 达到目标金额后，资金解冻
- 平台收取 5% 手续费（用于平台运营）
- 剩余资金转入发起人账户
- 发起人需上传资金使用凭证（医疗单据、发票）

**众筹失败**：
- 未达到目标金额，截止时间到
- 资金原路退回捐赠者
- 3 个工作日内到账
- 平台不收取任何费用

**资金公示**：
- 众筹页面实时显示资金使用情况
- 发起人上传医疗费用支出凭证
- 透明公开，接受监督

### 7.5 平台捐赠

**公益基金**：首页 → 爱心捐赠 → 平台公益基金

**捐赠用途**：
- 平台运营维护
- 紧急救助储备金
- 志愿者活动支持
- 公益宣传推广

**捐赠流程**：
1. 选择捐赠金额（10 元 / 50 元 / 100 元 / 自定义）
2. 完成支付
3. 获得电子捐赠证书

**资金公示**：
- 每月公开公益基金收支明细
- 每季度发布资金使用报告
- 接受社会监督
