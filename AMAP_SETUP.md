# 🔑 高德地图 API Key 申请指南

## 📝 申请流程

### 第一步：注册高德开放平台账号

1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 点击右上角 **"注册"**
3. 使用手机号注册并完成验证

### 第二步：创建应用

1. 登录后进入 [控制台](https://console.amap.com/)
2. 点击 **"我的应用"** → **"创建新应用"**
3. 填写应用信息：
   - **应用名称**: `PawLink宠物救助地图`
   - **应用类型**: 选择 `移动端`
   - **应用描述**: `宠物救助和领养应用的地图功能`

### 第三步：添加Key

1. 在创建的应用下，点击 **"添加Key"**
2. 填写Key信息：
   - **Key名称**: `PawLink Web服务Key`
   - **服务平台**: 选择 `Web端(JS API)`

### 第四步：配置Key权限

在Key管理页面，启用以下服务：

- ✅ **地图显示** (默认开启)
- ✅ **搜索服务** (需要 POI 搜索时)
- ✅ **地理编码** (地址解析，默认开启)

### 第五步：获取Key

复制生成的 **API Key**，格式类似：
```
4dxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

⚠️ **重要**: 请妥善保管API Key，不要提交到公共代码仓库！

---

## 🔐 安全配置

### IP白名单（可选）

为了提高安全性，可以设置IP白名单：

1. 在Key设置中，找到 **"安全设置"**
2. 添加服务器IP（如果需要）
3. 对于客户端应用，通常不限制IP

### 每日调用限制

- **新用户**: 每日免费额度 30,000 次
- **企业认证**: 每日免费额度 100,000 次
- **超额后**: 按调用次数计费

对于PawLink应用，预计每日调用量：
- 地图加载: 约 1,000 次/天
- 定位请求: 约 2,000 次/天
- **总计**: 约 3,000 次/天
- **结论**: 在免费额度内 ✓

## 📋 项目配置

获取到API Key后，需要在项目中配置：

### 1. 创建环境变量文件

在项目根目录创建 `.env` 文件：

```bash
# 高德地图API Key
AMAP_API_KEY=你的API_Key_在这里
```

### 2. 配置说明

API Key将用于：
- 加载高德地图JS API
- 地理编码（地址解析）
- 逆地理编码（坐标转地址）

---

## ✅ 验证配置

完成申请后，请验证：

1. 在浏览器访问：`https://webapi.amap.com/api/jsapi?v=2.0&key=你的Key`
2. 如果看到地图页面，说明配置成功
3. 如果看到错误页面，请检查Key是否正确
